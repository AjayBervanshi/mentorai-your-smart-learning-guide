# Skill Tracker & Learning App

A personalized learning and skill-tracking application designed to help users structure their learning journey, track their progress, build projects, and prepare for interviews.

## Features

- **Personalized Onboarding:** Tailor your learning experience by adding skills, current proficiency levels, goals, and daily time commitment.
- **Dashboard:** An intuitive overview of your learning journey. Track your current streak, total XP, overall progress, and access quick actions.
- **Structured Learning Paths:** Dynamic topic generation for each skill based on your proficiency level (Beginner, Intermediate, Advanced).
- **Interactive Quizzes:** Test your knowledge after each lesson with quizzes featuring a hint system, scoring, and detailed explanations.
- **Progress Tracking:** Detailed breakdowns of your overall and per-skill progress, completed topics, and areas for improvement.
- **Projects:** Unlock project templates tailored to your skill level as you progress, ranging from simple calculators to full-stack mini apps.
- **Interview Preparation:** Practice answering common interview questions tailored to your learned skills, complete with sample answers and expert tips.

## Tech Stack

- **Framework:** React 18, Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui (Radix UI)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router
- **State Management:** React Query, Context API

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```shell
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```shell
   cd <project-directory>
   ```
3. Install dependencies:
   ```shell
   npm install
   ```
   *Note: This project also supports `bun` (via `bun.lockb`).*

### Running the App

Start the development server:

```shell
npm run start-dev
```
*(where start-dev runs the vite command)*

### Building for Production

```shell
npm run build
```

### Running Tests

```shell
npm run test
```
